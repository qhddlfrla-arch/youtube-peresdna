# AI 페르소나 생성기

AI를 활용하여 창의적인 캐릭터와 인물(페르소나)을 자동으로 생성하고 관리하는 도구입니다.

## 🎯 주요 기능

### 1. 페르소나 생성
- **AI 자동 생성**: Gemini AI를 사용하여 독특하고 생동감 있는 페르소나 자동 생성
- **카테고리 선택**: 
  - 유튜버
  - 강사/교육자
  - 게임 캐릭터
  - 소설 인물
  - 영화 캐릭터
  - 브이로거
  - 팟캐스트 진행자
  - 비즈니스 전문가
  - 예술가
  - 운동선수
  - 역사적 인물 (현대적 해석)
  - 판타지 캐릭터
- **컨텍스트 설정**: SF, 판타지, 현대 등 원하는 배경 설정 가능
- **맞춤 특성**: 유머러스, 지적인, 낙천적 등 원하는 특성 지정 가능
- **다중 생성**: 한 번에 최대 5개의 페르소나 동시 생성

### 2. 페르소나 정보
각 페르소나는 다음 정보를 포함합니다:
- **기본 정보**: 이름, 설명, 나이, 성별, 직업
- **성격**: 5가지 주요 성격 특성
- **말투/어투**: 캐릭터의 독특한 언어 스타일
- **배경 이야기**: 2-3문장의 상세한 배경 설명
- **추가 특성**: 취미, 특기 등 다양한 세부 특성

### 3. 페르소나 관리
- **목록 보기**: 생성된 모든 페르소나를 한눈에 확인
- **상세 보기**: 각 페르소나의 모든 정보를 자세히 열람
- **편집**: 페르소나의 모든 속성을 직접 수정 가능
- **삭제**: 불필요한 페르소나 제거
- **AI 개선**: 기존 페르소나를 AI를 통해 더욱 발전시키기

### 4. 데이터 관리
- **자동 저장**: 모든 페르소나가 브라우저에 자동 저장
- **개별 내보내기**: 선택한 페르소나를 JSON 형식으로 다운로드
- **전체 내보내기**: 모든 페르소나를 한 번에 백업
- **전체 삭제**: 모든 데이터 한 번에 초기화
  - 선택한 챕터만 다운로드
  - 메타데이터 포함/제외 선택
- **별도 페이지**: 다운로드 전용 페이지에서 옵션 선택

### 6. 데이터 보존

## 🛠️ 기술 스택

- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS
- **AI**: Google Gemini AI (gemini-1.5-flash)
- **Build**: Vite
- **Deployment**: Vercel
- **API**: Google Gemini AI API (페르소나 생성)

## 📦 설치 및 사용 방법

### 1. 필수 요구사항
- Node.js 18 이상
- npm 또는 yarn
- Google Gemini API 키

### 2. 설치
```bash
# 저장소 클론
git clone https://github.com/qhddlfrla-arch/Youtube_analyze.git
cd Youtube_analyze

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

### 3. API 키 설정
1. [Google AI Studio](https://makersuite.google.com/app/apikey)에서 무료 API 키 발급
2. 앱 실행 후 설정 버튼(⚙️) 클릭
3. API 키 입력 및 저장

### 4. 사용 방법
1. **카테고리 선택**: 원하는 페르소나 유형 선택
2. **컨텍스트 입력** (선택): 특정 배경이나 세계관 설명
3. **생성 개수 선택**: 1-5개 중 선택
4. **특정 특성 입력** (선택): 원하는 성격이나 특징 입력
5. **페르소나 생성** 버튼 클릭
6. 생성된 페르소나 확인 및 편집

## 💡 사용 예시

### 예시 1: 유튜버 페르소나 생성
- **카테고리**: 유튜버
- **컨텍스트**: 게임 리뷰 채널
- **특정 특성**: 유머러스, 열정적, 솔직한
- **결과**: 게임 리뷰에 최적화된 유튜버 페르소나 생성

### 예시 2: 소설 캐릭터 생성
- **카테고리**: 소설 인물
- **컨텍스트**: 중세 판타지 세계, 마법사 길드
- **특정 특성**: 지적, 신비로운, 냉정한
- **결과**: 판타지 소설에 사용할 수 있는 마법사 캐릭터 생성

## 🔧 주요 기능 상세

### AI 페르소나 개선
기존 페르소나를 선택한 후 개선 요청을 입력하면 AI가 페르소나를 더욱 발전시킵니다:
- "더 유머러스하게 만들어줘"
- "배경 이야기를 더 상세하게 작성해줘"
- "말투를 더 격식있게 바꿔줘"

### 데이터 내보내기
- **JSON 형식**: 다른 프로젝트에서 재사용 가능
- **자동 백업**: 로컬 스토리지에 자동 저장
- **개별/전체 내보내기**: 선택적 백업 지원

## 🎨 커스터마이징

- 성격 특성 (쉼표로 구분)
- 말투/어투 설명
- 배경 이야기
- 추가 특성 (취미, 특기 등)
