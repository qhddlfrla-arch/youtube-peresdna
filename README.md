# 유튜브 영상 분석 AI 웹사이트

AI를 활용한 유튜브 영상 대본 분석 및 콘텐츠 기획 도구입니다.

## 🎯 주요 기능

### 1. 영상 분석
- **URL 입력**: 유튜브 영상 URL 입력으로 자동 분석
- **대본 추출**: 자동 자막 추출 및 텍스트 변환
- **AI 분석**: 
  - 핵심 키워드 추출
  - 영상 기획 의도 분석
  - 조회수 예측 및 성공 요인 분석
  - 대본 구조 분석 (타임스탬프 포함)

### 2. 콘텐츠 기획
- **아이디어 생성**: 분석 결과 기반 새로운 영상 아이디어 6개 자동 생성
- **키워드 맞춤**: 특정 키워드를 포함한 아이디어 생성 가능
- **카테고리별 분석**: 스토리텔링, 교육/정보, 엔터테인먼트 등 10개 카테고리 지원

### 3. 대본 생성
- **자동 대본 작성**: AI가 키워드와 영상 길이에 맞춰 대본 자동 생성
- **챕터 구조**: 자동 챕터 구성 (인트로, 본론, 결론 등)
- **챕터별 대본**: 각 챕터마다 상세한 대본 생성 가능
- **캐릭터 설정**: 영상 스타일에 맞는 캐릭터 자동 생성
- **다양한 스타일**: 
  - 스토리텔링 (브이로그)
  - 교육/강의형
  - 리뷰/비교형
  - 엔터테인먼트형
  - 다큐멘터리형

### 4. 대본 편집 및 관리
- **드래그 앤 드롭**: 챕터 순서 자유롭게 변경
- **실시간 편집**: 
  - 제목/부제목 수정
  - 대본 내용 직접 편집
  - 타임스탬프 수정
  - 캐릭터 이름/설명 변경
- **총 시간 계산**: 편집 중 실시간으로 총 영상 시간 표시
- **초기화 기능**: 원본 AI 추천 순서로 복구

### 5. 다운로드 기능
- **다양한 형식 지원**:
  - TXT (순수 텍스트)
  - JSON (데이터 구조)
  - Markdown (마크다운)
  - CSV (표 형식)
- **다운로드 옵션**:
  - 전체 대본 다운로드
  - 선택한 챕터만 다운로드
  - 메타데이터 포함/제외 선택
- **별도 페이지**: 다운로드 전용 페이지에서 옵션 선택

### 6. 데이터 보존
- **자동 저장**: 모든 분석 결과 및 생성된 대본 자동 저장
- **로컬 스토리지**: 브라우저 로컬에 저장 (24시간 보존)
- **새로고침 복원**: 페이지 새로고침해도 작업 내용 유지
- **수동 초기화**: 사용자가 원할 때만 데이터 삭제 가능

### 7. 오류 처리
- **상세 오류 메시지**: 
  - 오류 원인 자동 분석
  - 단계별 해결 방법 안내
  - 개발자 전달 정보 포함
- **오류 복사 기능**: 클릭 한 번으로 오류 정보 복사
- **오류 로그**: 최근 10개 오류 자동 저장

### 8. 광고 시스템
- **Google AdSense 통합**
- **다양한 광고 위치**:
  - 사이드바 광고 (좌/우)
  - 하단 고정 광고
  - 컨텐츠 사이 광고
- **광고 차단 감지**: 애드블록 사용 시 안내 모달

### 9. 보안 기능
- **복사 방지**: 콘텐츠 복사 차단 (입력 필드 제외)
- **우클릭 방지**: 컨텍스트 메뉴 비활성화
- **드래그 방지**: 이미지 및 텍스트 드래그 차단
- **개발자 도구 경고**: 콘솔 접근 시 경고 메시지

### 10. 사용자 경험
- **반응형 디자인**: 모바일/태블릿/데스크톱 최적화
- **로딩 애니메이션**: 분석 중 시각적 피드백
- **진행 상태 표시**: 각 작업 단계별 상태 표시
- **키보드 단축키**: ESC로 모달 닫기 등

## 🛠️ 기술 스택

- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS
- **AI**: Google Gemini AI (gemini-2.5-flash)
- **Build**: Vite
- **Deployment**: Vercel
- **APIs**: 
  - YouTube Data API (영상 정보)
  - YouTube Transcript API (자막 추출)
  - Google Gemini AI API (분석 및 생성)

## 📦 설치 방법

### 1. 필수 요구사항
- Node.js 18 이상
- npm 또는 yarn
- Google Gemini API 키
- (선택) YouTube Data API 키
- (선택) Google AdSense 계정

### 2. 프로젝트 설치

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

### 3. 환경 변수 설정

`.env.local` 파일 생성:

```env
GEMINI_API_KEY=your_gemini_api_key_here
```

### 4. 애드센스 설정 (선택사항)

다음 파일에서 `ca-pub-YOUR-ADSENSE-ID`를 본인의 애드센스 ID로 교체:

- `index.html`
- `components/AdSense.tsx`
- `components/SidebarAds.tsx`
- `components/FloatingAnchorAd.tsx`

### 5. 도메인 설정

`index.html`에서 `https://your-domain.com`을 본인의 도메인으로 교체

## 🔑 API 키 발급

### Google Gemini AI API

1. https://aistudio.google.com 접속
2. `Get API Key` 클릭
3. API 키 생성 및 복사
4. 웹사이트에서 API 키 입력

### YouTube Data API (선택사항)

1. https://console.cloud.google.com 접속
2. 프로젝트 생성
3. YouTube Data API v3 활성화
4. API 키 생성

- 사용자가 본인의 API 키를 입력하도록 설계됨
